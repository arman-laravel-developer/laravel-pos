Laravel Project Installation Guide

Prerequisites

  • XAMPP installed (or any local server environment like WAMP or MAMP)
  • PHP 8.1 or higher installed
  • Composer installed
  • MySQL database server

Installation Steps

  1. Clone the Laravel project repository from GitHub:
    2. git clone https://github.com/arman-laravel-developer/laravel-pos.git
  2. Navigate into the cloned project directory:
    3. cd repository
  3. Copy .env.example file to .env
    4. copy .env.example .env
  4. Configure your database settings on .env:
    5. DB_CONNECTION=mysql
 DB_HOST=127.0.0.1
 DB_PORT=3306
 DB_DATABASE=your_database_name
 DB_USERNAME=root
 DB_PASSWORD=
  5. Start your Apache and MySQL servers from the XAMPP Control Panel.
  6. Access phpMyAdmin by going to http://localhost/phpmyadmin in your browser.
  7. Create a new MySQL database with the same name as specified in your .env file (your_database_name).
  8. Import the SQL file manually using phpMyAdmin:
    1. Click on the newly created database.
    2. Click on the "Import" tab.
    3. Choose your SQL file.(database/laravel_pos.sql)
    4. Click "Go" to import the database.
  9. Install project dependencies using Composer:
    10. composer install
  10. Generate application key:
    11. php artisan key:generate
  11. Start the development server:
    12. php artisan serve
  12. Access the application in your browser:
    13. http://localhost:8000
  13. Admin Access:
    14. Admin Email: admin@gmail.com
    Admin Password: 12345678

Additional Configuration

All product and order routes are written in routes/admin.php file. All POS routes are written in routes/pos.php file.

All view files in POS are viewed from the pos_views/ directory

Conclusion

You have successfully cloned and installed the Laravel project with PHP 8.1 and MySQL database on your XAMPP server environment, and manually imported the database from an SQL file.
