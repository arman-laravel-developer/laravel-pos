Laravel Pos Project Installation Guide

Prerequisites

  • XAMPP installed (or any local server environment like WAMP or MAMP)
  • PHP 8.1 or higher installed
  • Composer installed
  • MySQL database server

Installation Steps

  1. Download the zip file containing the Laravel project.
  2. Extract the contents of the zip file to a directory on your local machine.
  3. Start your Apache and MySQL servers from the XAMPP Control Panel.
  4. Access phpMyAdmin by going to http://localhost/phpmyadmin in your browser.
  5. Create a new MySQL database with the same name as specified in your .env file (your_database_name).
  6. Import the SQL file manually using phpMyAdmin:
    1. Click on the newly created database.
    2. Click on the "Import" tab.
    3. Choose your SQL file.(database/laravel_pos.sql)
    4. Click "Go" to import the database.
  7. Configure your database settings:
    8. DB_CONNECTION=mysql
 DB_HOST=127.0.0.1
 DB_PORT=3306
 DB_DATABASE=your_database_name
 DB_USERNAME=root
 DB_PASSWORD=
  8. Import your MySQL database from the SQL file:
    9. The database directory of the project contains the database SQL file. (database/laravel_pos.sql)
  9. Navigate to the project directory using the command line.
  10. Start the development server:
    11. php artisan serve
  11. Access the application in your browser:
    12. http://localhost:8000
  12. Admin Access:
    13. Admin Email: admin@gmail.com
    Admin Password: 12345678

Additional

All product and order routes are written in routes/admin.php file. All POS routes are written in routes/pos.php file.

All view files in POS are viewed from the pos_views/ directory

Conclusion

You have successfully installed the Laravel project with PHP 8.1 and MySQL database from a zip file and imported your database from an SQL file.
